# FlowSync SaaS Landing Page

A modern and fully responsive SaaS landing page built with **React**, **Vite**, and **Tailwind CSS**. The project showcases a clean user interface, smooth navigation, responsive layouts, and professional design principles commonly used in modern software-as-a-service websites.

## Features

- Modern landing page with a compelling hero section
- Responsive navigation and layout
- Feature showcase section
- Professional pricing plans
- Customer reviews and testimonials
- Contact and call-to-action section
- Built with React components for maintainability
- Styled using Tailwind CSS
